# Contributor Covenant 行为准则

## 我们的承诺

为了营造一个开放、友善、尊重与协作的社区环境，作为贡献者和维护者，我们承诺让每个人都能在本项目中获得免于骚扰的参与体验，无论其年龄、体型、可见或不可见的残障、族裔、性别特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、宗教或性取向如何。

## 我们鼓励的行为

有助于建设积极社区环境的行为包括：

- 对他人保持尊重与善意
- 接受不同观点与经验
- 理性接收和提供建设性反馈
- 为社区整体利益做出决策
- 对使用者、贡献者和维护者保持同理心

## 不可接受的行为

以下行为不可接受：

- 使用带有性暗示或攻击性的语言或图像
- 人身攻击、侮辱性评论、公开羞辱
- 骚扰、跟踪或持续打扰他人
- 未经许可公开他人隐私信息
- 其他在专业协作环境中可被合理视为不当的行为

## 维护者职责

项目维护者有责任解释并执行我们的社区行为标准，并有权对任何他们认为不当、具有威胁性、冒犯性或有害的行为采取适当且公平的纠正措施。

维护者有权删除、编辑或拒绝与本行为准则不符的评论、提交、代码、Wiki 编辑、Issue 及其他贡献。

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公共场合代表本项目时的行为，包括但不限于：

- GitHub Issue
- Pull Request
- 讨论区
- 项目相关社区交流

## 举报方式

如果你遇到不当行为，可通过仓库维护者的 GitHub 联系方式私下沟通，或通过项目相关私密渠道反馈。请尽量避免在公开 Issue 中披露涉及个人隐私或冲突升级的信息。

## 执行说明

维护者将根据实际情况评估每一起事件，并在必要时采取提醒、警告、限制互动或拒绝参与等措施。

## 来源说明

本行为准则参考 `Contributor Covenant` 社区行为准则进行整理与本地化改写。
